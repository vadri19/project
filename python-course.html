Урок 1: Введение в Python

Python - это простой и мощный язык программирования. Он отлично подходит для начинающих.

Что вам понадобится:

  • Установленный Python (версия 3.6 или выше).
  • Текстовый редактор (VS Code, Sublime Text, Atom).

Первая программа: "Hello, World!"

Создайте файл `hello.py` и добавьте в него следующий код:

print("Hello, World!")

Запустите программу в терминале

Урок 2: Переменные и типы данных

Переменные используются для хранения данных. В Python есть несколько основных типов данных:

  • Целые числа (int): `10`, `-5`, `0`
  • Числа с плавающей точкой (float): `3.14`, `-2.5`
  • Строки (str): `"Hello"`, `'Python'`
  • Логические значения (bool): `True`, `False`

Пример использования переменных:

name = "Alice" age = 30 pi = 3.14 is_student = False print(f"Name: {name}") print(f"Age: {age}") print(f"Pi: {pi}") print(f"Is student: {is_student}")

Урок 3: Условные операторы (if, else, elif)

Условные операторы позволяют выполнять код в зависимости от определенных условий.

age = 20 if age >= 18: print("Вы совершеннолетний") else: print("Вы несовершеннолетний")

Урок 4: Циклы (for, while)

Циклы позволяют повторять определенный блок кода несколько раз.

Цикл for:

for i in range(5): print(i) # Выведет числа от 0 до 4

Цикл while:

count = 0 while count < 5: print(count) count += 1

Урок 5: Функции

Функции позволяют организовывать код в логические блоки и повторно использовать его.

def greet(name): print(f"Hello, {name}!") greet("Bob") # Вызовет функцию и выведет "Hello, Bob!"
